# spring-boot-microservices-practice

This is a practice project created to learn the basics of spring boot framework with microservices integration.

### Developer Instructions:
 ##### Requirements:
 - JDK 1.8+
 - Maven 3x
 
 ##### Build Using: 
 - `mvn package`
